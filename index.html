Setting Up Cursor AI

Get started with the most powerful AI-powered code editor

Download Cursor

https://cursor.sh
2

Install and Launch

Run the installer and follow the setup wizard. Cursor is built on VS Code, so it will feel familiar if you've used it before.

  • ‚úì Import your VS Code settings and extensions

  • Use Cmd/Ctrl + K to edit code inline

Step 2

GitHub Integration

Version control and collaboration made easy

1

Create GitHub Account

If you don't have one, sign up at github.com. GitHub is free for unlimited public and private repositories.

2

Install Git

Download and install Git from git-scm.com. Verify installation by running:

git --version
3

Configure Git

Set up your identity in Git. This information will be used for all your commits.

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
4

Create a Repository

Initialize Git in your project and create your first repository:

git init
git add .
git commit -m "Initial commit"
5

Push to GitHub

Create a new repository on GitHub, then link it to your local project:

git remote add origin https://github.com/username/repo.git
git branch -M main
git push -u origin main
üí° Tip: Use GitHub Desktop or Cursor's built-in Git features for a visual interface!
Step 3

Deploy with Netlify

Deploy your website in seconds with continuous deployment

1

Sign Up for Netlify

Go to netlify.com and sign up using your GitHub account for seamless integration.

2

Connect Your Repository

Click "Add new site" ‚Üí "Import an existing project" ‚Üí Select GitHub and choose your repository.

  • ‚úì Automatic builds on every push
  • ‚úì Preview deployments for pull requests
  • ‚úì Instant rollbacks if needed
3

Configure Build Settings

For a static HTML site, you typically don't need build commands. Just set the publish directory:

Build command: (leave empty)
Publish directory: . (or your build folder)
4

Deploy Your Site

Click "Deploy site" and Netlify will build and publish your website. You'll get a unique URL like:

https://your-site-name.netlify.app
5

Custom Domain (Optional)

Add your own domain in the "Domain settings" section. Netlify provides free HTTPS certificates automatically!

üöÄ Bonus: Set up form handling, serverless functions, and environment variables in Netlify's dashboard!
Step 4

Integrate Supabase

Add authentication, database, and real-time features to your app

1

Create Supabase Account

Visit supabase.com and sign up. Create a new project and note your API keys.

2

Get Your Credentials

In your Supabase dashboard, go to Settings ‚Üí API to find your:

  • ‚úì Project URL
  • ‚úì Anon/Public Key
  • ‚úì Service Role Key (keep secret!)
3

Install Supabase Client

Add the Supabase JavaScript library to your project:

<script src="https://cdn.jsdelivr.net/npm/@supabase/supabase-js@2"></script>
4

Initialize Supabase

Create a connection to your Supabase project:

const supabaseUrl = 'YOUR_SUPABASE_URL'
const supabaseKey = 'YOUR_ANON_KEY'
const supabase = window.supabase.createClient(supabaseUrl, supabaseKey)
5

Create Your First Table

In the Supabase dashboard, go to Table Editor and create a new table. Example:

Table: messages
Columns:
- id (int8, primary key, auto-increment)
- content (text)
- created_at (timestamp)
6

Query Your Database

Use simple JavaScript to interact with your database:

// Insert data
await supabase.from('messages').insert({ content: 'Hello!' })

// Read data
const { data } = await supabase.from('messages').select('*')
üîê Security: Don't forget to set up Row Level Security (RLS) policies in Supabase for production apps!
